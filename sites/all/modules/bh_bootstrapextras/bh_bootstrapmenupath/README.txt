INSTALLATION

To install the module:

	1.	Copy this module folder into your site's modules directory
	2.	Copy the jquery_update module [1] into your site's modules directory
	3.	Enable the jquery_update module at admin/modules
	4.	At admin/config/development/performance, set the jQuery version field
		to 1.7 or higher
	5.	Enable this module

[1] http://drupal.org/project/jquery_update: as of this writing, the development
	version of the module (7.x-2.x-dev) must be used to update jQuery to the
	required version (1.7)
